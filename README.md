# Laporan Praktikum Pemrograman Teknologi Mobile

Repository ini berisi dokumentasi dan laporan praktikum untuk mata kuliah **Praktikum Pemrograman Teknologi Mobile**. Seluruh aktivitas praktikum selama satu semester dicatat dan disusun dalam repository ini menggunakan sistem version control Git sehingga setiap perkembangan dan perubahan dapat terdokumentasi dengan baik.

Setiap praktikum disimpan dalam direktori terpisah yang diberi nama sesuai hal yang dikerjakan saat praktikum (Dasar Pemrograman dengan Kotlin dan seterusnya). Di dalam setiap direktori terdapat file **README.md** yang menjelaskan kegiatan praktikum yang dilakukan pada minggu tersebut, serta dapat dilengkapi dengan file tambahan seperti kode program, catatan, maupun screenshot hasil praktikum.

Struktur ini dibuat agar laporan praktikum tersusun secara sistematis, mudah dibaca, dan memudahkan proses evaluasi oleh dosen.
